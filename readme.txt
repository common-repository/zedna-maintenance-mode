Set you website under maintenance to lock content while development.

== Description ==

You can lock whole website by setting it under maintenance.

Features:

- Select roles who applies maintenance mode

- Set a redirect to login page

- Edit a maintenance message in wysiwyg editor including media

== Installation ==

1. Upload `zedna-maintenance-mode` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

`Can i set maintenance mode for Super admin?`

On single site yes, it's just administrator, but on multisite no as there is no need to have maintenancefor the biggest developer in da house.
